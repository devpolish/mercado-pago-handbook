## Mercado Pago Handbook

### Payment Methods

```ruby
  @client.get("/v1/payment_methods")
```

This returns all payment methods available in your country.

### Plans
----
##### Actions

> Retrieves information about a plan  
> GET /v1/plans/:id  

> Creates a new plan  
> POST /v1/plans  

> Updates a plan  
> PUT /v1/plans/:id
